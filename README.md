## Xenia Plymouth Theme

INSTALL

    1. Move to /usr/share/plymouth/theme/xenia or whatever.
    2. Edit 'xenia.plymouth' and replace @IMAGES@ with that path.
    3. $ plymouth-set-default-theme -R xenia
